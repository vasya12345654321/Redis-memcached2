## Задание 1. Кеширование

Кеширование помогает ускорить работу приложений и снизить нагрузку на сервер.

Примеры проблем, которые решает кеширование:

* Ускоряет загрузку часто запрашиваемых данных.
* Уменьшает количество обращений к базе данных.
* Снижает нагрузку на сервер при большом количестве пользователей.
* Ускоряет работу сайтов и приложений.
* Позволяет быстрее получать данные, которые редко изменяются.

Например, если на сайте часто показывается список популярных товаров, его можно сохранить в кеше и не обращаться к базе данных при каждом запросе.
